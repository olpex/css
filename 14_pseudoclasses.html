Pseudoclasses

:link - Selects unvisited links.

:visited - Selects visited links.

:hover - Selects elements when the mouse is over them.

:active - Selects elements when they are active.

:focus - Selects elements when they have focus.

:target - Selects elements that are the target of the current URL.

:enabled - Selects enabled elements.

:disabled - Selects disabled elements.

:checked - Selects checked elements.

:indeterminate - Selects indeterminate elements.

:root - Selects the root element of the document.

:nth-child(2) - Selects the second child element.

:nth-last-child(2) - Selects the second last child element.

:first-child - Selects the first child element.

:last-child - Selects the last child element.

:first-of-type - Selects the first element of a particular type.

:last-of-type - Selects the last element of a particular type.

:only-child - Selects elements that are the only child of their parent.

:only-of-type - Selects elements that are the only element of their type.

:empty - Selects elements that are empty.

:lang(en) - Selects elements with a language code of "en".
