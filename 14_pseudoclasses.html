Посібник з Псевдокласів CSS

:hover

Псевдоклас :hover застосовується, коли користувач наводить курсор на елемент.

Наведіть курсор на це посилання 

.hover-example a {
    text-decoration: none;
    color: #3498db;
    transition: color 0.3s;
}
.hover-example a:hover {
    color: #2ecc71;
}

:active

Псевдоклас :active застосовується, коли елемент активується (наприклад, натискається).


.active-example button {
    padding: 10px 20px;
    background-color: #e74c3c;
    color: white;
    border: none;
    cursor: pointer;
    transition: background-color 0.3s;
}
.active-example button:active {
    background-color: #c0392b;
}

:focus

Псевдоклас :focus застосовується, коли елемент має фокус.


.focus-example input {
    padding: 8px;
    border: 2px solid #bdc3c7;
    transition: border-color 0.3s;
    width: 100%;
    box-sizing: border-box;
}
.focus-example input:focus {
    border-color: #8e44ad;
    outline: none;
}

:first-child та :last-child

Псевдокласи :first-child та :last-child вибирають перший та останній дочірні елементи відповідно.

  • Перший елемент
  • Середній елемент
  • Останній елемент

.first-last-example ul li:first-child {
    color: #e67e22;
    font-weight: bold;
}
.first-last-example ul li:last-child {
    color: #e74c3c;
    font-weight: bold;
}

:nth-child()

Псевдоклас :nth-child() вибирає елементи на основі їх позиції серед сусідів.

  • Елемент 1
  • Елемент 2
  • Елемент 3
  • Елемент 4
  • Елемент 5
  • Елемент 6
  • Елемент 7
  • Елемент 8

.nth-child-example ul li:nth-child(odd) {
    background-color: #ecf0f1;
}
.nth-child-example ul li:nth-child(even) {
    background-color: #bdc3c7;
}

:not()

Псевдоклас :not() вибирає всі елементи, які не відповідають селектору всередині дужок.

Цей абзац не виділено.

Цей абзац виділено.


.not-example p:not(.highlight) {
    color: #95a5a6;
}
.highlight {
    color: #2c3e50;
    font-weight: bold;
}

:link та :visited

Псевдокласи :link та :visited стилізують непосещені та пройдені посилання відповідно.

Відвідати Example.com 

.link-example a:link {
    color: #2980b9;
}
.link-example a:visited {
    color: #8e44ad;
}

:disabled та :enabled

Псевдокласи :disabled та :enabled стилізують елементи форм на основі їх стану. 


.disabled-example button:disabled {
    background-color: #95a5a6;
    cursor: not-allowed;
}
.enabled-example button:enabled {
    background-color: #27ae60;
    cursor: pointer;
}

:checked

Псевдоклас :checked стилізує перемикачі або чекбокси, які вибрані.



Опція 3 

.checked-example input[type="checkbox"]:checked + label {
    color: #d35400;
    font-weight: bold;
}

Додаткові Псевдокласи для Вивчення

Хоча приклад охоплює багато популярних псевдокласів, ось декілька, які ви можете додатково вивчити:

  • :nth-of-type() – Вибирає nth-дочірній елемент певного типу.
  • :first-of-type – Вибирає перший дочірній елемент певного типу.
  • :last-of-type – Вибирає останній дочірній елемент певного типу.
  • :only-child – Вибирає елемент, який є єдиним дочірнім елементом свого батька.
  • :empty – Вибирає елементи, які не мають дітей.
  • :target – Стилізує цільовий елемент URL, що містить ідентифікатор фрагмента.

Не соромтеся додавати їх до прикладу, щоб розширити ваші знання!
